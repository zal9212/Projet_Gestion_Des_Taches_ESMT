# Roadmap Mini-Project

This folder contains the modularized version of the roadmap page originally in `django.html`.

- `index.html` – main HTML file for the roadmap.
- `css/style.css` – extracted stylesheet.
- `js/script.js` – extracted JavaScript for interactivity, including theme toggling (dark/light).

Open `roadmap/index.html` in a browser to view the roadmap interface.
